# 创建自己的 Crate

安装`cargo-edit`工具，他扩展了`cargo`的功能

```powershell
# 项目根目录下下

cargo install cargo-edit
```

- cargo add toml：将最新的`toml` Crate安装到当前项目中，并自动写入当前目录下的`Cargo.toml`文件中。
