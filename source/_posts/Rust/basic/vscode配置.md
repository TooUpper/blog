# 安装必要插件

- rust-analyzer ：Rust 插件
- crates ：crate 相关插件
- Even Better TOML ：TOML 语法高亮



# 一些非必要但是好用的 crate

- cargo-edit: 该工具扩展了Cargo，允许您通过从命令行修改Cargo.toml文件来添加、删除和升级依赖项。

```shell
# 任意目录下
cargo install cargo-edit
```
