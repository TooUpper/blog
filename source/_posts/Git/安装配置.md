# 安装与配置

## SSH

```she
git config --global user.name 'kay.wang'

git config --global user.email 'cn.kay.wang@gmail.com'
```

## 生成本地密钥

```shell
ssh-keygen -t rsa -C 'cn.kay.wang@gmail.com'
```

下面一路回车即可

# 答疑

### -bash: ssh-keygen: command not found

未安装SSH

```shell
sudo pacman -S openssh
```
